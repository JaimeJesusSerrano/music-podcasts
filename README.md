## How to start the project in localhost 
Executing the next commands:
- `npm ci` # To install dependencies
- Check the `.env` file
- `npm run start`

## How build it 
Executing the next commands:
- `npm ci` # To install dependencies

### Dev mode
- `npm run build:dev`

### Production mode
- `npm run build`

## How to start tests
Executing the next command:
- `npm run test`
